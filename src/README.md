# @alphabetiq/Colors

Easy to use color primitives based on Material Design Color Palette.

## Installation

Install with `yarn add @alphabetiq/colors` or `npm i -s @alphabetiq/colors`

## Usage

_Example in React_

```
import { red, orange, blue } from "@alphabetiq/colors";

const colors = {
    primary: red[900],
    secondary: red[200],
    info: blue[600]
}
```
